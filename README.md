## 💭 About

***ttyrpg*** is a rogue-like teletype terminal game similar to Dwarf Fortress in style.

## Installation

### Source

Using git, clone the repository:

```shell
git clone https://github.com/crhowell3/ttyrpg.git
cd ttyrpg
```

Using cargo, install:

```shell
 cargo install --path .
 ```
